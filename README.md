# SkillDev

Шаблон проекта Vue 3 + TypeScript с готовой инфраструктурой. Настроены ESLint, Prettier. Написан Dockerfile и настроен nginx.conf. Представлен одной главной страницой ввиде счетчика.

## Установка для разработки

Клонировать репозиторий:

```bash
   git clone https://github.com/user/skilldev.git
   cd skilldev
```

Установить зависимости:

```bash
   npm install
```

Запустить локально:

```bash
   npm run dev
```
Приложение будет доступно по адресу: http://localhost:3005

## Команды

```bash
npm run dev     # Запуск разработки
npm run build   # Сборка продакшн
npm run lint    # Проверка кода
npm run format  # Форматирование
```
